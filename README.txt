To execute Project7WB.cpp make sure ALL files included within the folder Project6WB are in the directory that you attempt use the following commands on.

Next open up the command prompt/terminal and do the following: 
Note: Verify that you see all contents of Project7WB when performing the command "ls" on that directory, if you see them this means you can proceed.

CheckeredTriangle.cpp
ColorCubeFlyby.cpp

To compile CheckeredTriangle.cpp type: g++ CheckeredTriangle.cpp -o Triangle -lglut -lGLU -lGL
To compile ColorCubeFlyby.cpp type: g++ ColorCubeFlyby.cpp -o Cube -lglut -lGLU -lGL

Following this,

To execute CheckeredTriangle.cpp type: ./Triangle
To execute ColorCubeFlyby.cpp type: ./Cube
